### Skeleton Slim 3 + Gulp ###

el proposito de este proyecto es tener un lienzo limpio para programar y maquetar un proyecto pequeno/mediano, modificar el fichero gulpfile.js y config.php.

ejecutar:

```
  composer install
  
  npm install && npm start

```

### backend ###
  * Slim 3 Framework

  * ORM idiorm

  * Composer

### frontend ###
  * gulp

  * stylus

  * nunjucks

  * browsersync
